LeetCode 1529 - Bulb Switcher (IV)

There is a room with n bulbs (numbered from 0 to n-1) arranged in a row from left to right.  Initially, all the bulbs are turned off.

Your task is to obtain the configuration represented by target where target[i] is '1' if the i-th bulb is turned on and is '0' if it is turned off.

You have a switch to flip the state of the bulb, a flip operation is defined as follows:

Choose any bulb (index i) of your current configuration.

Flip each bulb from index i to n − 1.

When any bulb is flipped it means that if it is 0 it changes to 1 and if it is 1 it changes to 0.

Return the minimum number of flips required to form target.

target: "10111"

3

Initial configuration: "00000".
Flip from the first bulb: "00000" ➔ "11111"
Flip from the second bulb: "11111" ➔ "10000"
Flip from the third bulb: "10000" ➔ "10111"
We need at least 3 flip operations to form target.

target: "101"

3

"000" ➔ "111" ➔ "100" ➔ "101"

target: "001011101"

5

"000000000" ➔ "001111111" ➔ "001000000" ➔ "001011111" ➔ "001011100" ➔ "001011101"

Dayu Wang's Solution

class Solution {

    public int minFlips(String target) {

        int count = 0, swtch = 0;

        for (int i = 0; i < target.length(); i++) {

            if (target.charAt(i) - '0' != swtch) {

                count++;

                swtch = 1 - swtch;

            }

        }

        return count;

    }

}

Time complexity: O(n)

Space complexity: O(1)
